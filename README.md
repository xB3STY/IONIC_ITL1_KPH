# IONIC_ITL1

## 1. Aufgabenstellung: Start einer Demo App (z. B. tabs oder sidemenu)
---

## Tabs

**Erstellung:**  
`ionic start myTabsApp tabs --type=angular`

**Start:**  
`cd myTabsApp`

**Danach:**  
`ionic serve --lab`

**Ordner:**  
`myTabsApp`

## Sidebar (SideMenu)

**Erstellung:**  
`ionic start myApp sidemenu --type=angular`

**Start:**  
`cd myApp`

**Danach:**  
`ionic serve --lab`

**Ordner:**  
`myApp`
